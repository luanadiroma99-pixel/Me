🔬 Métabolisme Ajusté (Hypothyroïdie)

MB thyroïde : ~1 210 kcal/j

TDEE (entraînement 5-6j) : ~1 900 kcal/j

Apport cible : 1 450-1 500 kcal/j

⚠️ Phase 1 (running) : 1 500 kcal pour l'énergie. Phase 2 : 1 400-1 450.

✨ Pourquoi ce mix est parfait pour toi

Running : Brûle-graisse #1, améliore ta VO2max

Aquabike : Cardio sans impact, drainage, anti-cellulite

Renfo poids du corps : Tonification sans prise de masse

Corde à sauter : HIIT naturel, ~15 kcal/min

Yoga : Réduit le cortisol (ennemi #1 thyroïde), souplesse

Sauna : Récupération, détox, anti-inflammation

⚕️ Spécificités Thyroïde

✓ Lévothyrox à jeun, 30 min avant tout

✓ Pas de calcium/fer/soja 4h après

✓ Sauna APRÈS l'entraînement, jamais avant

✓ Hydratation ++ (sauna + running = perte d'eau)

✓ Sélénium + zinc + vit D = tes 3 alliés

🔑 Pourquoi tes régimes passés ont échoué

L'hypothyroïdie ralentit ton métabolisme de ~12%. Les déficits agressifs déclenchent le mode « survie ». Notre approche : déficit doux + activités variées qui boostent le métabolisme sans le stresser. Le yoga réduit le cortisol, le running et l'aquabike brûlent les graisses, le renfo sculpte sans volume.
